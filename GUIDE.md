# 試験作問テンプレート 利用ガイド (AI 向け)

このディレクトリは、科目によらず使い回せる定期試験の作問テンプレート一式である。
振動工学の試験 (親ディレクトリ `../`) を雛形に、科目固有の中身を抜いて
プレースホルダ化したもの。新しい科目の試験を作るときは、このディレクトリを
科目フォルダにコピーして使う。

## ワークフロー

1. 利用者と相談しながら、まず **md ファイル** (`question.md` / `answer.md`) で
   問題文・数式・模範解答を固める。
2. 固まった md を、AI が **html** (`question.html` / `answer.html` /
   `answer_sheet.html`) に変換する。html は印刷用の最終成果物。
3. ブラウザで html を開き、A4 で PDF に印刷する (MathJax が CDN から数式を描画)。

md が情報の正本、html は体裁を与えた清書、という役割分担。
内容を直すときは md を直してから html に反映する。

## ファイル一覧

| ファイル | 役割 |
|---|---|
| `question.md` / `question.html` | 問題用紙 |
| `answer.md` / `answer.html` | 模範解答 (途中式つき) |
| `answer_sheet.html` | 解答用紙 (生徒記入用、マス目)。md は持たず html を直接編集する |
| `assets/` | 図 (png など) を置く |
| `GUIDE.md` | このガイド |

## プレースホルダ

`{{...}}` をすべて科目ごとの値に置き換える。代表的なもの:

| プレースホルダ | 例 | 説明 |
|---|---|---|
| `{{科目名}}` | 振動工学 | 科目名 |
| `{{試験名}}` | 前期中間試験 | 試験の名前 |
| `{{年度}}` | 2026年度 | 年度 |
| `{{クラス記号}}` | 5ME-VE | クラス・コース記号。出席番号欄にも使う |
| `{{試験時間}}` | 90分 | |
| `{{配点}}` | 100点 | 試験全体の配点 |
| `{{大問配点}}` | 30点 | 各大問の配点 |
| `{{小問配点}}` | 5点 | 各小問の配点 |

## 解答の数値表記の規約

- 数値解答は **有効数字3ケタの指数表記** $X.YZ \times 10^{W}$ で書く。
  - 例: $12345 \Rightarrow 1.23 \times 10^4$、$-0.9896 \Rightarrow -9.90 \times 10^{-1}$、
    $3 \Rightarrow 3.00 \times 10^0$。
- この規約は問題用紙の「注意事項」と解答用紙のマス目 (仮数 × 10^指数) に対応する。
- 模範解答の最終解は `\boxed{...}` で囲む。
- 単位は `\text{ ... }` で数式内に入れる (例: `\boxed{4.00 \times 10^{1} \text{ Hz}}`)。

## md → html 変換ルール

### question.md → question.html

| md | html |
|---|---|
| `# 見出し (科目名・試験名)` | `.exam-header` のタイトルへ。注意事項は固定テンプレへ |
| `## 問N [..点]` | `<h2>問N [..点]</h2>` |
| 大問直後のリード文 | `<p class="q-intro">...</p>` |
| `**(n)** 本文` | `<p class="q-item"><strong>(n)</strong> 本文</p>` |
| `![](./assets/x.png)` | `<div class="figure"><img src="./assets/x.png" alt="説明"></div>` |
| `$$ ... $$` ブロック | `<div class="dm">$$ ... $$</div>` |
| `---` | `<hr>` |

### answer.md → answer.html

| md | html |
|---|---|
| `# 見出し` | `<h1>...模範解答</h1>` と `<p class="meta">{{年度}}</p>` |
| `## 問N [..点]` | `<h2>` |
| `### (n) 見出し` | `<h3>` |
| 説明文 | `<p>...</p>` |
| `$$ ... $$` ブロック | `<div class="dm">$$ ... $$</div>` |

### 共通の注意

- インライン数式は `$...$`、別行立ては `$$...$$` (MathJax 設定済み)。
- `$$` は式と同じ行に置かず、`.dm` の中で式とは別の行に書く
  (`<div class="dm">$$` で改行 → 式 → 改行 `$$</div>`)。
  一部レンダラが同一行の `$$式$$` をブロック数式と認識しないため。
- html は CDN の MathJax と Google Fonts を読むので、描画にはネット接続が要る。

## 解答用紙 (answer_sheet.html) の組み方

解答用紙は md を経由せず、html を直接編集する。問題の小問数と形式に合わせて
解答欄を機械的に組む。

共通ルール:

- 大問1つ = `<div class="section">` 1つ。
- 各大問の **最初の行だけ** `<span class="q-major">問N</span>`、
  2行目以降は空の `<span class="q-major"></span>`。

解答欄は問題の形式に応じて3種類から選ぶ (テンプレに問1〜問3の例がある):

| 種類 | クラス | 用途 | 1行あたりの小問数 |
|---|---|---|---|
| A. 指数表記 | `.ans-wrap` (`.ans-mantissa` × 10^ `.ans-exp`) | 数値計算問題 | 最大2 |
| B. 選択・記号 | `.ans-choice` | 選択問題 (記号を記入) | 最大2 |
| C. 自由記述 | `.ans-free-item` + `.ans-free` | 記述・計算過程・作図 | 1 |

- A・B は 1 行 (`.ans-row`) に小問 (`.ans-item`) を **最大2つ**。小問が奇数で
  余ったら空の `<div class="ans-item"></div>` で右側を埋める。
  例: 小問6個なら 2個ずつ3行、5個なら 2+2+1 で3行。
- C は幅が広いので **1 行 1 小問**。枠の高さは
  `<div class="ans-free" style="min-height: 45mm;">` のように個別調整してよい。
- 形式は大問ごとに混在してよい (問1=指数表記、問2=選択、問3=記述 など)。
  問題用紙の各小問の形式に合わせること。

## 図 (assets) の扱い

- 図は `assets/` に置き、`./assets/ファイル名` で参照する。
- WSL/Windows 由来の `*:Zone.Identifier` が付くことがあるので、不要なら消す。
- 振動工学の例図 (`nejiri.png` / `Scratch.png`) は他科目では使わないので、
  新科目では差し替えるか削除する。

## 体裁の前提 (変更しないことが多い箇所)

- 用紙 A4、余白 20mm、本文フォント Noto Serif JP、本文 10.5pt。
- ページ番号はフッタ中央に `- n -`。
- これらは各 html の `<style>` 内に自己完結で書かれている (外部 CSS なし)。
  科目をまたいで体裁をそろえたいので、原則ここはいじらない。
