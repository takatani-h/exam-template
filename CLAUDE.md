# CLAUDE.md — 試験作問テンプレート

このディレクトリは、科目によらず使い回す定期試験の作問テンプレート一式である。
詳しい仕様・変換ルールは **`GUIDE.md`** に書いてある。作業前に必ず参照すること。

## ワークフロー (要点)

1. まず **md** (`question.md` / `answer.md`) で問題文・数式・模範解答を固める。
   md が情報の正本。
2. 固まった md を **html** (`question.html` / `answer.html` /
   `answer_sheet.html`) に変換する。html は A4 印刷用の清書。
3. 内容修正は md を直してから html に反映する。

## 守ること

- `{{...}}` のプレースホルダ (`{{科目名}}` `{{試験名}}` `{{大問配点}}` ほか) を
  すべて科目の値に置き換える。一覧は `GUIDE.md`。
- 数値解答は有効数字3ケタの指数表記 $X.YZ \times 10^{W}$。模範解答の最終解は
  `\boxed{...}`、単位は `\text{ ... }` で式内に入れる。
- ブロック数式の `$$` は式と別行に置き、`.dm` でラップする。
- 各 html の `<style>` (A4・Noto Serif JP・体裁) は科目をまたいでそろえるため、
  原則いじらない。
- 図は `assets/` に置き `./assets/ファイル名` で参照。例図 (`nejiri.png`
  `Scratch.png`) は他科目では差し替え/削除する。

## 使い方

このディレクトリを新しい科目フォルダにコピーし、md を埋めてから html 化する。
完成済みの実例は `../` (振動工学の前期中間試験) を参照。
